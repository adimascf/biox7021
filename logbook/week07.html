Is there any difference between depth?

Based on the plots above, it seems that 20x depth consistanly perfomed worst than the other two depths, both on SNP and INDEL. I further performed a statistical test to confirm whether sequencing depth leads to differences in performance. The data is not normally distributed, instead, they are clustered right at the top (skewed).Therefore, I used a non parametric Kruskal–Wallis test, which ranks the data rather than relying on their raw values. This method is also robust in the small size dataset.

Code 
import scipy.stats as stats

def kruskall_test(dataframe, score_type):
  for vartype in ["SNP", "INDEL"]:
    print(f"Testing for {vartype}")

    subset = dataframe[dataframe["VAR_TYPE"] == vartype]

    # Extract F1 scores for each depth
    d100 = subset[subset["depth"] == "100x"][score_type]
    d50 = subset[subset["depth"] == "50x"][score_type]
    d20 = subset[subset["depth"] == "20x"][score_type]

    # perform the test
    stat, p_value = stats.kruskal(d100, d50, d20)

    print(f"Kruskal-Wallis Statistic: {stat:.4f}")
    print(f"P-value: {p_value:.4e}")

So, based on the test, different depth has a significant impact on both SNP and INDEL F1 scores. Those depths are not perfomring equally. But where is the difference? To identify which depth different (already seems obvious from the plot 😅), I performed a post hoc test, comparing each depth pairwise to determine which ones contribute to the observed differences.

Code 
import scikit_posthocs as sp

def post_hoc(dataset, score):
  for vartype in ["SNP", "INDEL"]:
    print(f"Testing for {vartype}")

    subset = dataset[dataset["VAR_TYPE"] == vartype]

    p_values = sp.posthoc_dunn(subset, val_col=score, group_col='depth', p_adjust='bonferroni')

    print(p_values.round(5))

frames = []
for p in map(Path, input_files):
    df = pd.read_csv(p, sep=r"\s+", thousands=",")
    # extract wildcard from Path
    df["sample"] = p.stem.split(".")[0]
    df["trimmer"] = p.parts[-2]
    # keep only relevant columns
    frames.append(df[["sample", "trimmer", "num_seqs", "sum_len"]])

df_stats = pd.concat(frames).reset_index(drop=True)

# create a baseline dataframe of untrimmed counts
baseline = df_stats[df_stats["trimmer"] == "untrimmed"][["sample", "num_seqs", "sum_len"]]
baseline = baseline.rename(columns={"num_seqs": "raw_seqs", "sum_len": "raw_len"})

# merge baseline back to the main dataframe
df_comparison = df_stats.merge(baseline, on="sample")

# calculate discarded amounts
df_comparison["base_discarded"] = df_comparison["raw_len"] - df_comparison["sum_len"]
df_comparison["percent_base_discarded"] = (df_comparison["base_discarded"] / df_comparison["raw_len"]) * 100

df_comparison["base_discarded"] = df_comparison["raw_len"] - df_comparison["sum_len"]
df_comparison["percent_base_discarded"] = (df_comparison["base_discarded"] / df_comparison["raw_len"]) * 100

# just give the proper name
df_comparison["trimmer"] = df_comparison["trimmer"].replace({
    "dorado": "dorado_trim",
    "porechop_abi_nocheck": "porechop_abi_nosplit"
})

# aggregate results to see which trimmer discards the most on average
summary = df_comparison.groupby("trimmer")[["base_discarded", "percent_base_discarded"]].mean().sort_values("percent_base_discarded", ascending=False).round(2)
