Week 14: Visualisation exploration and quality based tools assessment

Overview

I made pointplot figures for the F1 metrics. I ran into a weird issue while creating them, but managed to find a workaround.

I also explored different visualisation approaches, including pointplot, stripplot + boxplot, and barplot, as well as using either the median or mean as the point/bar height for each assessment metric we generated.

I still do not see any clear winner or loser among the trimming tools tools, or any tool that consistently performed better than the untrimmed dataset.

Selected and ran a representative seq of quality based preprocessing tools with their respective paramter sets. The, I compiled and plotted the resulting results.

Weekly meeting and action items (27-05-2026)

  • Issue with logit scalling on the y axis and average calculation in the pointplot/barplot.
  • My current workaround
  • Median vs mean? I personally prefer using the median because most of our data is highly skewed (towards 1 or towards 0). Also, median is resistant to outliers. However, in some cases we can observe “pattern” more clearly using the mean compared to no pattern on median.
  • Normalised NGA50 value
  • For sanity check, based on rasusa downsampling log all samples after quality filtering were above 99.96x (~100x) depth.
  • Filtlong default (which prefers longer and higher-qscore reads) is the best

from itables import show
from pathlib import Path
import pandas as pd

def get_genome_size(sample_name):
    """Parses the fasta index file to sum the total genome size for a sample."""
    fai_path = Path(f"../results/reference/{sample_name}.fa.fai")
    try:
        with open(fai_path, "r") as f:
            # The second column (index 1) in a .fai file is the sequence length
            return sum(int(line.split("\t")[1]) for line in f)
    except FileNotFoundError:
        print(f"Warning: .fai file not found for {sample_name}. Returning NaN.")
        return np.nan


df = pd.read_csv("../results/tables/assess/assembly-alltrim/metrics/quast_compiled_metrics.csv")

# Normalise NGA50 per species
df['genome_size'] = df['sample'].apply(get_genome_size)
df['NGA50_norm'] = df['NGA50'] / df['genome_size']
df.sort_values(by=["trimmer", "sample", "model", "depth"], inplace=True)

show(df, scrollX=True, scrollY="800px")
