Week 15: Which combination of tools give the best result?

Overview

I assessed combination of adapter trimming tools and quality based preprocessing tools, where the trimming step was pefromed before quality trimming and/or filtering. I also compiled resources usage metrics for each tool. Overall, `filtlong, remained the best performning tool across variant calling and genome assembly metrices.

Weekly meeting and action items (03-06-2026)

  • porechop_abi discard or porechop_abi split?
  • for untrimmeed (not processed trough adapter trimming step), do we chose fastq from basecaller --trim or --no-trim?. In the previous quality preprocessing only assessment, I used FASTQ files generated with the default trimming enabled. In this combined assessment, however, untrimmed refers to FASTQ files generated using Dorado with the --no-trim option.

Tools selection

Residual adapter processing tools

  • untrimmed -> The baseline control, fastq files from --no-trim basecalling process

  • dorado trim -> The ndustry standard. If purely based on performance, fastplong is arguably a better choice here (and much faster)

  • barbell_default -> Performed better than the --maximize option overall. In particular, barbell_max showed a significant drop in mean NGA50 at 20× depth.

  • porechop_abi_discard -> Performed much better overall compared to the --no_checkoption in both variant calling and assembly metrics. While it performed close to the default parameters (--split), porechop_abi_discard showed great F1 scores on HAC data.

Quality based preprocessing tools

With the help of gemini, i chose:

  • filtlong_default -> Clearly the best performing tool overall. It has the highest F1 score and among the lowest assembly errors tools.

  • chopper_extract100 -> It resulted in fewer assembly errors overall while maintaining competitive F1 scores. chopper_extract1000 is also a reasonable choice, but chopper_extract100 generally produced fewer false positives.

  • seqkit_1000 -> Achieved pretty good F1 scores and was among the best performing tools in terms of minimising assembly errors, while also maintaining NGA50 well.

  • unprocessed -> Negative control

I also assessed a tool that performs both adapter trimming and quality based preprocessing by default. Without adding much complexity to the pipeline, I included fastplong_all (length ≥ 100 and quality ≥ 15 for SUP, ≥ 10 for HAC) as a standalone option in the pipeline.

I make the pipeline so that the tool selection is flexible. If we decide to include additional tools or replace any of the currently assessed tools, I can easily update the pipeline.

import sys
import pandas as pd
import matplotlib.pyplot as plt
import seaborn as sns
import numpy as np
from typing import List

# colourblind-friendly palette from colour universal design (CUD)
# https://jfly.uni-koeln.de/color/
# https://github.com/mbhall88/cud

named_colors = {
    "black": "#000000",
    "orange": "#e69f00",
    "skyblue": "#56b4e9",
    "vermilion": "#d55e00",
    "bluish green": "#009e73",
    "yellow": "#f0e442",
    "blue": "#0072b2",
    "reddish purple": "#cc79a7",
}
cud_palette = list(named_colors.values())

def cud(n: int = len(cud_palette), start: int = 0) -> List[str]:
    remainder = cud_palette[:start]
    palette = cud_palette[start:] + remainder
    return palette[:n]


input_csv = "../results/tables/assess/call-combo/metrics/combo_variant_summary.csv"

sns.set_theme(style="whitegrid")

pr_df = pd.read_csv(input_csv)

metrics = ["F1_SCORE"]
vartypes = ["SNP", "INDEL"]
models = ["sup", "hac"]
estimators = ["mean", "median"]
hue_order = ["100x", "50x", "20x"]

for y in metrics:
    for est in estimators:
        # Calculate OVERALL performance to sort from best (highest) to worst (lowest)
        if est == "mean":
            combo_perf = pr_df.groupby("combo")[y].mean()
            est_func = 'mean'
        else:
            combo_perf = pr_df.groupby("combo")[y].median()
            est_func = 'median'

        order = combo_perf.sort_values(ascending=False).index.tolist()

        fig, axes = plt.subplots(
            nrows=len(vartypes),
            ncols=len(models),
            figsize=(15, 10),
            dpi=300,
            sharex=True,
            sharey=False,
        )

        for r, vartype in enumerate(vartypes):
            for c, model in enumerate(models):
                ax = axes[r, c]
                df = pr_df.query("VAR_TYPE == @vartype and model == @model").copy()

                cap = 0.99999
                df.loc[:, y] = df[y].apply(lambda v: cap if v > cap else v)
                yticks = [0.01, 0.1, 0.5, 0.8, 0.9, 0.99, 0.999, 0.9999, cap]
                yticklabels = [f"{yval:.2%}" for yval in yticks]

                if not df.empty:
                    sns.pointplot(
                        data=df, x="combo", y=y, hue="depth",
                        order=order, hue_order=hue_order,
                        palette=cud(len(hue_order), start=2),
                        ax=ax,
                        dodge=0.3,
                        errorbar=('ci', 95),
                        capsize=0.1,
                        err_kws={'linewidth': 1.5},
                        estimator=est_func
                    )

                ax.set_yscale("logit", nonpositive="clip")
                ax.set_yticks(yticks)
                ax.set_yticklabels(yticklabels)

                ax.set_ylabel(f"{vartype} F1 score ({est})")

                if r == 0:
                    ax.set_title(f"{model} model", fontsize=14)
                else:
                    ax.set_title("")
                ax.tick_params(axis="x", labelrotation=45, labelsize=10)

                ax.set_xlabel("")

                ax.set_xticks(range(len(order)))
                ax.set_xticklabels(order, rotation=45, ha="right", rotation_mode="anchor", fontsize=11)

                ax.xaxis.grid(True, linestyle='--', color='lightgrey', zorder=0)

                # Legend on the top right plot
                if r == 0 and c == 1:
                    if ax.get_legend() is not None:
                        ax.legend(title="Depth", bbox_to_anchor=(1.05, 1), loc='upper left')
                else:
                    if ax.get_legend() is not None:
                        ax.get_legend().remove()

        fig.tight_layout()
        fig.show()

import sys
from pathlib import Path
import pandas as pd
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns
from typing import List

# colourblind-friendly palette from colour universal design (CUD)
named_colors = {
    "black": "#000000",
    "orange": "#e69f00",
    "skyblue": "#56b4e9",
    "vermilion": "#d55e00",
    "bluish green": "#009e73",
    "yellow": "#f0e442",
    "blue": "#0072b2",
    "reddish purple": "#cc79a7",
}
cud_palette = list(named_colors.values())

def cud(n: int = len(cud_palette), start: int = 0) -> List[str]:
    remainder = cud_palette[:start]
    palette = cud_palette[start:] + remainder
    return palette[:n]

def get_genome_size(sample_name):
    """Parses the fasta index file to sum the total genome size for a sample."""
    fai_path = Path(f"../results/reference/{sample_name}.fa.fai")
    try:
        with open(fai_path, "r") as f:
            return sum(int(line.split("\t")[1]) for line in f)
    except FileNotFoundError:
        print(f"Warning: .fai file not found for {sample_name}. Returning NaN.")
        return np.nan

df = pd.read_csv("../results/tables/assess/assembly-combo/metrics/combo_quast_compiled_metrics.csv")

# Normalise NGA50 per species
df['genome_size'] = df['sample'].apply(get_genome_size)
df['NGA50_norm'] = df['NGA50'] / df['genome_size']

sns.set_theme(style="whitegrid")
models = ["sup", "hac"]
hue_order = ["100x", "50x", "20x"]

plot_types = ["pointplot", "stripplot"]
estimators = ["mean", "median"] 
scales = ["linear", "logit"] # Added scales array

out_dir = Path('../results/figures/assess/assembly-combo/metrics/')
out_dir.mkdir(parents=True, exist_ok=True)

for est in estimators:

    if est == "mean":
        combo_perf = df.groupby("combo")["NGA50_norm"].mean()
        est_func = np.mean
    else:
        combo_perf = df.groupby("combo")["NGA50_norm"].median()
        est_func = np.median

    order_abs = combo_perf.sort_values(ascending=False).index.tolist()

    # if 'unprocessed-untrimmed' in order_abs:
    #     order_abs.remove('unprocessed-untrimmed')
    #     order_abs.append('unprocessed-untrimmed')

    for p_type in plot_types:
        for scale in scales: # Nested scale loop
            fig, axes = plt.subplots(nrows=1, ncols=len(models), figsize=(14, 5.5), dpi=300, sharex=True, sharey=False)

            for c, model in enumerate(models):
                ax = axes[c]
                df_sub = df.query("model == @model").copy()

                if not df_sub.empty:

                    cap = 0.9999

                    # Apply logit cap ONLY if plotting logit scale
                    if scale == "logit":
                        df_sub['NGA50_norm'] = df_sub['NGA50_norm'].apply(lambda v: cap if v >= cap else (0.00001 if v <= 0 else v))

                    if p_type == "pointplot":
                        sns.pointplot(
                            data=df_sub, x="combo", y="NGA50_norm", hue="depth",
                            order=order_abs, hue_order=hue_order,
                            palette=cud(len(hue_order), start=2),
                            ax=ax, dodge=0.3, errorbar=('ci', 95), capsize=0.1,
                            err_kws={'linewidth': 1.5}, estimator=est_func
                        )
                    elif p_type == "stripplot":
                        sns.boxplot(
                            data=df_sub, x="combo", y="NGA50_norm", hue="depth",
                            order=order_abs, hue_order=hue_order,
                            palette=cud(len(hue_order), start=2),
                            ax=ax, fliersize=0, gap=0.1
                        )
                        sns.stripplot(
                            data=df_sub, x="combo", y="NGA50_norm", hue="depth",
                            order=order_abs, hue_order=hue_order,
                            palette=cud(len(hue_order), start=2),
                            ax=ax, alpha=0.6, dodge=True, linewidth=0.5, edgecolor="black", legend=False
                        )

                # Dynamic Axis Scaling and Zoom
                if scale == "logit":
                    ax.set_yscale("logit", nonpositive="clip")
                    ax.set_ylim(0.5, 0.999)

                    # Custom ticks for the zoomed logit region
                    yticks = [0.5, 0.6, 0.7, 0.8, 0.9, 0.99, 0.999, cap]
                    yticklabels = [f"{yval:.2%}" for yval in yticks]
                    ax.set_yticks(yticks)
                    ax.set_yticklabels(yticklabels)
                elif scale == "linear":
                    ax.set_yscale("linear")
                    ax.set_ylim(-0.05, 1.05)

                # Text Formatting
                ax.set_ylabel(f"NGA50 (Norm) [{scale.title()} | {est.title()}]" if c == 0 else "")
                ax.set_title(f"NGA50 (Normalised) - {model} model")
                ax.set_xlabel("")

                ax.set_xticks(range(len(order_abs)))
                ax.set_xticklabels(order_abs, rotation=45, ha="right", rotation_mode="anchor", fontsize=11)
                ax.xaxis.grid(True, linestyle='--', color='lightgrey', zorder=0)

                # Legend handling
                if c == 1 and ax.get_legend() is not None:
                    ax.legend(title="Depth", bbox_to_anchor=(1.05, 1), loc='upper left')
                elif ax.get_legend() is not None:
                    ax.get_legend().remove()

            fig.tight_layout()

            # Output dynamically based on scale, estimator, and plot type
            new_output_path = out_dir / f"assembly_nga50_normalised_{scale}_{p_type}_{est}.png"
            fig.show()
